# 应急科普小游戏《急救小先锋》设计文档

## 1. 项目概述

### 1.1 项目名称
《急救小先锋》（First Aid Hero）

### 1.2 项目目标
开发一款寓教于乐的应急科普小游戏，通过互动式学习让儿童和青少年掌握基本急救技能，培养自救互救意识。

### 1.3 目标用户
| 用户群体 | 优先级 | 特点 |
|---------|--------|------|
| 6-12岁儿童 | 最高 | 需要简单直观的操作和有趣的视觉引导 |
| 13-15岁中学生 | 高 | 可以理解更复杂的急救知识和挑战机制 |
| 家庭用户 | 中 | 亲子共同学习，增进家庭安全意识 |

## 3. 游戏核心玩法

### 3.1 玩法模式

#### 模式一：剧情闯关模式
- **主线故事**：玩家扮演"急救小先锋"，接到救援中心的紧急任务，前往各种场景处理急救事件
- **场景设计**：温馨卡通风格的2D场景，包含客厅、厨房、操场、郊外等

#### 模式二：模拟操作模式
| 操作类型 | 交互方式 | 示例 |
|----------|----------|------|
| 点击按压 | 快速点击屏幕模拟胸外按压 | CPR场景 |
| 拖拽冲击 | 拖拽操作模拟腹部冲击 | 海姆立克场景 |
| 步骤点击 | 按顺序点击操作按钮 | AED使用场景 |

#### 模式三：知识问答模式
- **选择题**：四选一，答对获得分数
- **判断题**：常见急救误区判断
- **排序题**：急救步骤按正确顺序排列

#### 模式四：收集养成模式
- **道具收集**：完成关卡收集急救用品
- **成就系统**：达成目标解锁成就徽章
- **角色升级**：积累经验值升级

### 3.2 关卡设计体系

#### 第一篇章：生命基础急救（MVP核心）

##### 1. 心肺复苏CPR关卡

**成人CPR关卡**
- 学习目标：掌握成人胸外按压和人工呼吸操作
- 操作步骤：
  1. 判断意识：轻拍双肩呼唤
  2. 呼救：点击"拨打120"按钮
  3. 判断呼吸：观察胸廓起伏5-10秒
  4. 胸外按压：快速点击按压区域，频率100-120次/分钟，深度5-6cm
  5. 人工呼吸：仰头抬颏→捏住鼻子→吹气
  6. 循环操作：30次按压+2次人工呼吸

**儿童CPR关卡**
- 按压深度：约5cm
- 手法：掌根按压

**婴儿CPR关卡**
- 按压深度：约4cm
- 手法：两根手指按压
- 双人施救比例：15:2

**AED使用关卡**
- 操作步骤：
  1. 开机：点击电源按钮
  2. 贴电极片：拖拽电极片到正确位置（右上胸、左乳头外侧）
  3. 分析心律：等待机器分析，所有人离开
  4. 除颤：点击放电键
  5. 恢复CPR：立刻继续按压

##### 2. 海姆立克急救关卡

**成人/大龄儿童气道梗阻**
- 操作步骤：
  1. 站在患者身后，双臂环抱腰腹
  2. 一手握拳（拳心抵住上腹部肚脐上方）
  3. 另一手抓拳，快速向内上方冲击5次
  4. 重复直到异物咳出

**1岁以下婴儿呛噎**
- 操作步骤：
  1. 俯卧手臂上，头略低
  2. 两肩胛骨之间拍背5次
  3. 未排出翻转婴儿
  4. 两根手指胸部冲击5次
  5. 循环操作

**自身呛噎自救**
- 倚靠固定硬物（椅背、桌角）
- 上腹用力向内顶压

**昏迷异物梗阻处理**
- 平躺，开放气道
- 手指抠出可见异物
- 立刻开始CPR

## 4. 交互设计

### 4.1 操作方式
- **点击**：选择选项、确认操作、模拟按压
- **拖拽**：移动物品、包扎操作、模拟冲击
- **滑动**：场景切换、翻页查看信息

### 4.2 反馈机制
- **视觉反馈**：正确操作显示绿色√，错误操作显示红色×并提示正确做法
- **音效反馈**：成功音效、错误提示音、心跳声（CPR场景）
- **进度提示**：实时显示关卡进度、得分、剩余时间

### 4.3 学习辅助
- **提示系统**：操作困难时可查看步骤提示
- **回放功能**：错误操作后可观看正确示范动画
- **知识卡片**：完成关卡后解锁相关知识点卡片

## 9. 风险与注意事项

### 9.1 医学准确性风险
- **风险**：急救知识必须准确，错误信息可能误导用户
- **应对**：所有急救知识需由专业医护人员审核
- **验证**：开发过程中邀请急救培训师参与评审

### 9.2 游戏化平衡风险
- **风险**：过于娱乐化可能降低知识严肃性
- **应对**：保持教育性优先，娱乐性辅助
- **验证**：用户测试时评估知识掌握程度

### 9.3 操作误导风险
- **风险**：游戏操作与真实操作存在差异
- **应对**：每个关卡结束时强调"本游戏仅为教学，实际急救请遵循专业指导"
- **验证**：在显著位置添加免责声明
